# Futuristic SaaS

Next Generation Software Services Platform

## Overview
A fully responsive, modern, futuristic SaaS frontend built directly with:
- Next.js 15 (App Router)
- React 19
- Framer Motion
- Tailwind CSS v4
- Lucide React Icons

## Features
- **Dark Futuristic Theme**: Custom neon and green palette design system
- **Responsive**: Full mobile layout and sticky navbar with blur support
- **Animations**: Advanced scrolling animations utilizing Framer motion

## Getting Started

1. `npm install`
2. `npm run dev`
3. Hit `localhost:3000` in browser to see the experience!
